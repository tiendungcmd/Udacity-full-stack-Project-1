# Run local
``` shell
npm run start
```
Access url in browser `http://localhost:3000/api/images?name=<name-of-image>&width=<width-of-image>&height=<height-of-image>`

Check image exist in folder `process`
# Check Quality
To run unit test:
``` shell
npm run test
```
To eslint:
``` shell
npm run lint
```
To run prettier;
``` shell
npm run prettier
```
